Prototypes

There are no classes in Javascript, but it uses prototypes to allow object oriented programming. An object's prototype is a link to another object, allowing inheritance from the linked object.

Constructor Function

The most used way in JavaScript to construct prototype chains is the constructor function, which is a special function for creating and initializing a new object.

'new' keyword

When using a constructor function to create a new object, the keyword 'new' must be used before the function name.

The 'new' keyword looks at the prototype property of the constructor function we are calling, and uses that to construct a brand new object which it passes to the constructor function, and automatically returns it as the result of that function (if nothing else is returned).

The following is an example of how to use prototypes
